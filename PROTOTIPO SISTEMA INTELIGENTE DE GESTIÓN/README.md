# 🍫 ChocoInventory

**ChocoInventory** es un prototipo de plataforma web diseñada para pequeños y medianos negocios que necesitan una gestión inteligente de su inventario.
Este sistema permite registrar productos, controlar el stock, recibir alertas y predecir la demanda futura utilizando datos históricos.

## ✨ Características Principales

- **📊 Dashboard Interactivo**: Visualiza el estado actual de tu negocio con KPIs y gráficos en tiempo real.
- **📦 Gestión de Inventario**: Crea, edita y elimina productos fácilmente.
- **🚨 Alertas de Stock Bajo**: El sistema te avisa automáticamente cuando un producto necesita reabastecimiento.
- **🔄 Registro de Movimientos**: Control total de entradas (compras) y salidas (ventas).
- **🔮 Predicción de Demanda**: Algoritmo simple que analiza el historial de salidas para sugerir futuros pedidos.
- **📱 Diseño Responsivo**: Interfaz moderna y adaptable a dispositivos móviles.

## 🛠️ Tecnologías Usadas

Este prototipo ha sido construido utilizando tecnologías web estándar, sin dependencias complejas de backend:

- **HTML5**: Estructura semántica.
- **CSS3**: Estilos modernos con variables y diseño responsivo (Flexbox/Grid).
- **JavaScript (ES6+)**: Lógica del negocio y manipulación del DOM.
- **Chart.js**: Librería para visualización de datos (Gráficos).
- **Font Awesome**: Iconografía.
- **Google Fonts**: Tipografía 'Outfit'.
- **LocalStorage**: Persistencia de datos en el navegador del usuario.

## 🚀 Cómo Usar

1.  Clona este repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en tu navegador web de preferencia (Chrome, Edge, Firefox).
3.  ¡Listo! No necesitas instalar servidores ni bases de datos.

## 📝 Notas del Prototipo

- Los datos se guardan en el `localStorage` de tu navegador. Si borras la caché o cambias de dispositivo, los datos se perderán.
- La predicción de demanda es una simulación basada en un promedio móvil simple para fines demostrativos.
